Site Selector Setup Instructions

This program is only compatible with Windows.
__________________________________________________________________

1. Download the entire zip folder from GitHub to your desktop.

2. Open the folder

3. Find the Site Selector Shortcut file.

4. Move the Site Selector shortcut and the folder its in to your desktop.

5. Delete the zip folder

6. Now youre all set, start using Site Selector
